## Identity Rules

- `Role title` is mandatory.
- `Role name` is optional.
- `Role name` must be unique among active roles unless explicitly marked as alias, transition, or retired identity.
- `Canonical header` must match this file.
- If this file conflicts with role memory or role charter, this file wins and the mismatch should become a reconciliation task or feedback candidate.
